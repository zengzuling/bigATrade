# WeChat Daily Review Template

This template is for WeChat public-account stock recap articles that need search visibility, reader retention, comment interaction, and soft conversion.

The article should not read like a raw result report. It should read like a trading-learning recap:

`market emotion -> main sector -> core stock verification -> tracked stock separation -> tomorrow scenarios -> learning/conversion`

## Title Options

### Conservative

`核心个股/板块 + 今日A股复盘：强弱分层后，明日策略怎么定？`

### More viral

`核心个股涨停回封！真正该看的不是涨幅，而是分歧后谁敢接`

### Default recommendation

`核心个股/板块继续发酵：今日A股复盘与明日强势股处理计划`

## Search Keywords

Use 4 to 8 relevant keywords naturally in the title, opening paragraph, subheadings, and closing:

- `A股复盘`
- `今日复盘`
- `明日策略`
- `盘前计划`
- `短线情绪`
- `涨停复盘`
- `强势股`
- active sectors such as `CPO`, `PCB`, `半导体`, `小金属`, `商业航天`
- tracked stock names and codes

## Poster Copy Skeleton

Main title:

`核心个股涨停回封！`

Subtitle:

`分歧换手后再确认，今天这批推荐股表现分层`

Market sentence:

`今日A股情绪明显回暖/出现分歧/弱势修复，主线资金集中在XX方向。`

Short stock labels:

- `核心股: 涨停回封 / 达标验证 / 分歧承接`
- `补涨股: 跟随主线 / 有溢价 / 看延续`
- `趋势股: 放量走强 / 资金活跃 / 看承接`
- `容量股: 大成交震荡 / 看关键位`
- `偏弱股: 弱势观察 / 先看修复`

Closing line:

`结论：核心票验证预判；明天不追第一口，重点看分歧后的承接。`

## Article Skeleton

Opening hook:

`今天A股情绪明显回暖/分歧加大/弱势修复，XX方向继续活跃。我们重点跟踪的XX完成涨停回封/达标验证，XX和XX也给出溢价。今天这篇复盘，重点讲三个问题：强势股分歧后怎么看、核心票明天怎么处理、主线方向还能不能延续。`

Subheading 1:

`一、今日A股复盘：情绪到底强在哪里？`

Body:

`Use index performance, market breadth, turnover, hot sectors, and short-term emotion to define the day. Do not list data only. Convert the data into a clear judgment: repair, continuation, divergence, or broad weakness.`

Subheading 2:

`二、核心个股：涨停回封/达标验证说明什么？`

Body:

`Use the strongest tracked stock. Explain the chain: pre-market or previous-day plan -> intraday move -> divergence -> support -> close. The key sentence should teach a method, for example: 真正重要的不是涨停本身，而是分歧后资金还愿不愿意接。`

Subheading 3:

`三、5只推荐股表现：谁主动，谁跟随，谁偏弱？`

Body:

`Group the tracked stocks by market role: core validation, sector extension, trend support, capacity divergence, weak observation. Use real change percentage, gain from recommendation, and turnover only where they change the decision.`

Subheading 4:

`四、为什么说这不是事后诸葛亮？`

Body:

`Explain what was judged before the move and how today's market verified or rejected it. This paragraph builds trust and makes the article educational instead of promotional.`

Closing paragraph:

`复盘不是事后解释涨跌，而是盘前建立剧本、盘中验证资金、盘后修正策略。想系统学习盘前推演、盘中处理和盘后复盘的朋友，可以点我头像，私信我。`

## Tomorrow Plan Skeleton

Subheading:

`五、明日策略：高开不追，回踩看承接`

Scenario 1:

`For the strongest stock: if it opens too high or one-word limit-up, do not chase the first impulse. Watch seal strength, opening volume, whether the first break is supported, and whether it can quickly recover or reseal.`

Scenario 2:

`For sector-extension stocks: watch whether the main sector continues. If the sector keeps leading and the stock holds above the latest close or key support, it can stay on the observation list. If sector emotion weakens, lower expectations.`

Scenario 3:

`For trend/capacity stocks: watch the real price levels: latest close, intraday high, intraday low, buy range, target, and stop-loss. Use these levels as scenario boundaries, not absolute instructions.`

Scenario 4:

`For weak stocks: first watch repair and stabilization. If they cannot recover key levels or continue weak while the market is strong, treat them as risk-first names.`

Closing line:

`明天重点不是乱追强，而是看强势股分歧后的承接、补涨股的板块延续、弱势股的修复质量。`

`你更想看我明天重点拆CPO，还是小金属？留言多的方向，我盘前单独写。`
